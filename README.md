# SCUT Alumni America — By-Laws Revision Project

This repository manages the formal revision of the By-Laws of **South China University of Technology Alumni, America**. The current By-Laws were adopted in December 2002.

## How to Use This Repository

1. Read the current By-Laws in [current_version.md](current_version.md)
2. Review [PROCESS.md](PROCESS.md) to understand the formal amendment procedure and timeline
3. Browse [examples/](examples/) for bylaws and analysis of 10 comparable organizations — start with [examples/README.md](examples/README.md) for a prioritized list of governance recommendations
4. Work through the pre-drafting research in [planning/](planning/) — see [planning/README.md](planning/README.md) for a guide to the directory structure and how to use each file
5. Contribute to [proposal/rationale.md](proposal/rationale.md) by documenting issues and proposed changes
6. The working draft lives in [proposal/draft.md](proposal/draft.md)
