# SCUT Alumni America — By-Laws Revision Project

This repository manages the formal revision of the By-Laws of **South China University of Technology Alumni, America**. The current By-Laws were adopted in December 2002.

---

## Repository Structure

```
By-Law/
├── README.md                  ← This file
├── PROCESS.md                 ← Amendment procedure (per Article VIII) and project timeline
├── current_version.md         ← Current By-Laws as adopted on December 10, 2002
├── current_version.html       ← Original HTML source
├── planning/                  ← Pre-drafting research and policy analysis (see planning/README.md)
│   ├── README.md              ← Guide to the planning directory structure and file types
│   ├── 01-vision-and-purpose/
│   ├── 02-leadership-structure/
│   ├── 03-election-and-appointment/
│   ├── 04-membership-and-eligibility/
│   ├── 05-decision-process/
│   ├── 06-california-law-compliance/
│   ├── 07-financial-management/
│   ├── 08-meetings-and-notices/
│   ├── 09-digital-operations-and-communications/
│   ├── 10-standing-and-special-committees/
│   ├── 11-conflict-of-interest/
│   ├── 12-dissolution/
│   └── 13-federal-irs-501c7-compliance/
└── proposal/
    ├── rationale.md           ← Issues identified; justification for proposed changes
    └── draft.md               ← Working draft of the proposed new By-Laws
```

---

## How to Use This Repository

1. Read the current By-Laws in [current_version.md](current_version.md)
2. Review [PROCESS.md](PROCESS.md) to understand the formal amendment procedure and timeline
3. Work through the pre-drafting research in [planning/](planning/) — see [planning/README.md](planning/README.md) for a guide to the directory structure and how to use each file
4. Contribute to [proposal/rationale.md](proposal/rationale.md) by documenting issues and proposed changes
5. The working draft lives in [proposal/draft.md](proposal/draft.md)
